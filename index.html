Intro to Programming--- The Basics of the Web and HTML


HTML stands for Hypertext Markup Language
HTML is the main type of document found on the web



Types of files  that can be found on the web are the following:
Plain text, HTML, Images, Videos and Music



Major pieces of the web-

You
Your pc & browser
The internet
Servers


HTTP is the main protocol of the web.
Servers are the computers that host files that make up the web.
The "internet" is the world's largest computer network.
Browser is a program that allows you to display files found on the web.
(examples--IE, Google Chrome, Mozilla Firefox, Safari)


HTML is made of:

Text content  (what you see)
Markup (what it looks like)
References to other documents (e.g. images & videos)
Links to ther pages



Text will show up on one continuous line unless you apply
Marker to start a new line.


Tag:  HTML tags are always contained within angled brackets.
Tags will have an opening  and closing tag. "Void" don't have
closing tags.  The em is a "void" tag



Element: refer to everything within a set of opening and closing tags.
Inline elements include:  a, span, br, img and strong
Block elements include: p and div
The p block element creates an invisible box or block around the text
which also can have height and width



Attribute: Is a property of a HTML element.
